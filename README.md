# Summary
- ホームページに雪を降らせたり、イチョウやカエデの葉っぱを降らせたりするためのライブラリ


# Howto
1. HTML内に以下を設置
-  id="kaede"
-  内部に降らせるimageタグを設置
- #kaedeタグには、hidden属性を設置（読み込み完了まで非表示にするため）
```
<div id="kaede" hidden="hidden">
  <img src="sample.png" />
</div>
```

2. src/main.jsを読み込み（ページ内どこでも設置可）
